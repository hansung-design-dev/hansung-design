# 고객에게 보낼 디버깅 요청 메시지 (템플릿)

---

안녕하세요.

결제 과정에서 문제가 발생한 것으로 확인되어, 정확한 원인 파악을 위해 개발자 도구 콘솔 화면을 캡처해주시면 감사하겠습니다.

## 📋 캡처 방법 (간단 버전)

1. **결제 페이지에서 F12 키를 누르세요** (또는 마우스 우클릭 → "검사")
2. 상단의 **"Console"** 탭을 클릭하세요
3. **F5 키**로 페이지를 새로고침하세요
4. **결제하기 버튼을 클릭**하세요
5. **전체 콘솔 화면을 캡처**해서 보내주세요
   - 특히 🔍로 시작하는 로그들과 빨간색 에러 메시지가 중요합니다

## 📸 캡처 예시

다음과 같은 로그들이 보여야 합니다:
- `🔍 [결제 페이지 초기화] 브라우저 및 환경 정보:`
- `🔍 [결제 버튼 클릭] Step 1: openTossWidget 함수 호출됨`
- `🔍 [위젯 초기화] Step 1: initializeTossWidget 함수 시작`
- 등등...

에러가 있다면 빨간색 메시지가 나타납니다.

## 📧 함께 보내주실 정보

1. 브라우저 종류 및 버전 (Chrome, Edge 등)
2. 운영체제 (Windows 10, Windows 11, Mac 등)
3. 문제 발생 시점 (결제 버튼 클릭 후 모달이 뜨기 전인지, 모달은 뜨는데 위젯이 안 뜨는지 등)

캡처해주시면 빠르게 문제를 해결하겠습니다.

감사합니다.
